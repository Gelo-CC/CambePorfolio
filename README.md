# Enterprise Systems Suite

## POS • LMS • IMS Dashboard

Modern landing page showcasing integrated business systems:

- **POS** (Point of Sale): Real-time sales & invoicing
- **LMS** (Learning Management): Courses & student tracking  
- **IMS** (Inventory Management): Stock & orders

### Tech Stack
- Frontend: HTML5, Tailwind CSS, Vanilla JS
- Responsive design, interactive modals, smooth animations

### Quick Start
1. Open `index.html` in browser
2. No setup required - fully static
